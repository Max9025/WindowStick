# WindowStick

**WindowStick** — это небольшая утилита на C# (WinForms), которая позволяет закреплять выбранные окна поверх всех остальных (Always On Top).  

Приложение работает как обычное окно, так и из системного трея. При выходе утилиты все закреплённые окна сбрасываются (если они ещё живы).

---

## 🔹 Основные возможности

- Просмотр всех открытых окон в системе.  
- Закрепление окна поверх всех с помощью чекбокса.  
- Отмена закрепления окна.  
- Управление через системный трей:  
  - Левый клик — показать/скрыть главное окно.  
  - Контекстное меню (Open / Exit).  
- Корректный сброс TopMost для живых окон при выходе утилиты.  
- Иконка в трее встроена как Embedded Resource (не требует отдельного файла ICO).  
- Поддержка работы с окнами, которые скрыты в системном трее (через перебор всех окон процесса).  

---

## 🔹 Требования

- Windows 10+  
- .NET Framework 4.8  
- Visual Studio 2022 / 2019 (или другой IDE с поддержкой WinForms)  

---

## 🔹 Установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/Max9025/WindowStick.git
```

2. Откройте проект в Visual Studio.

3. Постройте проект и запустите ```WindowStick.exe```.

---

## 🔹 Использование

1. Выберите окно из списка.
2. Поставьте галочку «Always On Top» (чекбокс).
3. Окно закреплено и будет поверх всех.
4. Чтобы отменить закрепление, снимите галочку.
5. Для выхода используйте контекстное меню трея → ```Exit```.

---

## 🔹 Примечания

- Сброс TopMost работает для живых окон. Если окно было закрыто и открыто заново — это новое окно.
